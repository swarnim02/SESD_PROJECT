# Lost and Found Management System with Reward Feature

A web application to help users report lost items, post found items, and submit claims to recover belongings with optional reward system.

## Features
- User authentication (register/login)
- Report lost items with details
- Post found items
- Submit and manage claims
- Optional reward declaration
- Admin moderation panel
- Search and filter items

## Technology Stack
- **Backend:** Node.js + Express.js
- **Frontend:** React
- **Database:** MySQL/PostgreSQL
- **Auth:** JWT + Bcrypt
